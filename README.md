# FASTA Toolkit 🧬

A simple bioinformatics toolkit written in Python for analyzing DNA sequences from FASTA files.

## Features

This toolkit can:

- Read DNA sequences from a FASTA file
- Calculate sequence length
- Calculate GC content
- Generate reverse complement
- Convert DNA to RNA
- Handle multiple sequences in a FASTA file

## Requirements

Python 3.x

## Usage

Run the program using:

python main.py sample.fasta

## Example FASTA File

>Sequence_1
ATGCGTAGCTAGCTAGCTAG

>Sequence_2
GGCTAGCTAGCGATATCGC

## Example Output

Sequence: Sequence_1  
Length: 20  
GC Content: 50%  
Reverse Complement: CTAGCTAGCTACGCAT  
RNA: AUGCGUAGCUAGCUAG
